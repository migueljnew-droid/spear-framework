# SPEAR Examples

## Example Walkthroughs

These examples show SPEAR in action across different project types.

### Quick Start: Adding User Authentication

A complete walkthrough of one SPEAR cycle for adding JWT authentication to a Node.js API.

**Files:**
- `auth-feature/spec/prd.md` — The product requirements
- `auth-feature/spec/architecture.md` — Architecture decisions
- `auth-feature/plan/phase-1.md` — Implementation plan
- `auth-feature/audit/security-report.md` — Security audit findings
- `auth-feature/ratchet/retrospective.md` — What we learned
